# 💰 My-Bookkeeping-App

一个基于 Python 和 FreeSimpleGUI 开发的轻量化桌面记账工具，支持数据本地持久化存储功能。

---

## ✨ 功能特性

- **简洁界面**：采用 FreeSimpleGUI 构建，操作直观。
- **数据管理**：支持记账条目的添加、查看与删除。
- **持久化存储**：数据自动以 JSON 格式保存至本地 `data.txt`，防止丢失。

## 🚀 快速上手

### 1. 环境准备
确保你的电脑已安装 Python 3.8 或更高版本。

### 2. 安装依赖
克隆仓库到本地并安装必要的 GUI 库：
```bash
git clone [https://github.com/H2Log1/My-Bookkeeping-App.git](https://github.com/H2Log1/My-Bookkeeping-App.git)
cd 仓库名
pip install FreeSimpleGUI
```
